# Common Directory Reference

This directory is a reference to the shared common resources.

## Actual Location

The actual `common/` directory is located at:
```
D:\Projects\Native\RDX\RDC-Agent-Frameworks\debugger\common\
```

## Contents

The common directory contains:
- `AGENT_CORE.md` - Core agent definitions and rules
- `agents/` - Shared agent configurations
- `config/` - Platform capabilities and validation scripts
- `docs/` - Documentation
- `hooks/` - Shared hooks
- `knowledge/` - Shared knowledge base
- `project_plugin/` - Project plugin templates
- `skills/` - Shared skills

## Usage

Before using this platform, ensure the common directory is properly set up and run:
```bash
python common/config/validate_binding.py --strict
```
